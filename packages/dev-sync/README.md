# @ggez/dev-sync

Development-time editor and game synchronization utilities for Web Hammer.

## Install

```bash
bun add @ggez/dev-sync
```

## Exports

- `@ggez/dev-sync`: shared registry types and helpers
- `@ggez/dev-sync/node`: Node-side registry persistence helpers

Use this package to coordinate editor and local game runtime state during development.
